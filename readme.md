## FULL HTMLベースで作られたマンガエディタです。
MITライセンスですので、再配布、商用問題ありません。ライセンスをお読みください。
軽量です。1Mバイトもありません。windows/macとも動作します。chromeで動作確認しています。

## アプリの内容
- ページにコマを配置し、コマに任意の画像を流し込み、吹き出しとセリフを並べてマンガのページを作ることができます。
- 複数ページにも対応できます。
- 吹き出しは各種あります。融合もできます。
- セリフはアウトライン化して手書き風変形も可能。
- 集中線などのエフェクトもあり。
- 下書きモードあり。
- メモ書きモードあり。
- セーブ・ロード、別HTMLにコピー＆ペーストできます。

## 使い方
- マンガページエディタのzipを解凍し、index.htmlをクリックしてください。
- MANUAL.htmlまたは画面右上の「？」ヘルプ表示です。
- サンプルだよ.manga.jsonはサンプルセーブファイルです（スクリーンショットのモノです）

# マンガページエディタ by やまもりやもり
(c) 2026 やまもりやもり  

## FULL HTMLベースで作られたマンガエディタです。
- 作者: [やまもりやもり](https://yamamori-yamori.github.io/mypage/)
- 外部サーバー・CDN不要。完全ローカル動作。
- ブラウザでHTMLファイルを開くだけで動作します。Windows / macなどプラットフォームを問いません。
- MITライセンスですので、再配布、商用問題ありません。ライセンスをお読みください。  

## アプリの内容
- ページにコマを配置し、コマに任意の画像を流し込み、吹き出しとセリフを並べてマンガのページを作ることができます。
- 複数ページにも対応できます。
- コマの枠線はグリッド吸着できます。斜めにもグリッド引けます。
- 吹き出しは各種あります。融合もできます。
- セリフはアウトライン化して手書き風変形も可能。
- 集中線などのエフェクトもあり。
- 下書きモードあり。
- メモ書きモードあり。
- セーブ・ロード、別画面で開いたこのアプリ間でコピー＆ペーストできます。

## 使い方
- 右の「Releases」よりダウンロードしたmangaeditのzipを解凍し、index.htmlをクリックしてください。
- zip中のMANUAL.html クリックまたは、画面右上の「？」を押すとヘルプ表示です。
- 「サンプルだよ.manga.json」はサンプルセーブファイルです（スクリーンショットのモノです）
- 「いまある画像にちょっと吹き出しを付けたいだけ」という用途では、最初の紙を決める画面に直接画像をドロップしてください。

※セキュリティソフトのCrowdStrikeがあるとJavascriptを誤検知してzipが開けないかもしれません。その場合はpowershellのExpand-Archiveコマンドをご使用ください。
